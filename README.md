## 🧠 Sobre o Projeto

Este projeto foi desenvolvido para automatizar consultas no SAP (transação ES32), utilizando **Python + SAP GUI Scripting**, com foco em:

✔ Redução de tempo operacional  
✔ Padronização de análises técnicas  
✔ Aumento de produtividade  
✔ Minimização de erros manuais  

---

## ⚡ Problema de Negócio

Processos manuais no SAP exigem:

- Navegação repetitiva
- Alto tempo de consulta por unidade consumidora
- Risco de erro humano
- Baixa escalabilidade

---

## 💡 Solução Desenvolvida

Aplicação desktop com interface gráfica (Tkinter) que:

- Automatiza navegação no SAP
- Extrai dados críticos da instalação
- Organiza informações em formato estruturado
- Apresenta resultados em interface amigável

---

## 📊 Impacto Gerado

🚀 Redução significativa do tempo de consulta  
📈 Aumento de produtividade operacional  
🧠 Padronização do processo de análise  
⚙️ Automação de tarefas repetitivas  

---

## 🔍 Dados Capturados

O sistema realiza leitura automatizada de:

- Instalação / UC  
- Tipo de Instalação  
- Classificação  
- Parceiro de Negócios  
- Nome do Cliente  
- Endereço  
- Status da Instalação (com indicador visual)  
- Categoria de Atendimento  
- Fase de Ligação  
- Dados de Contrato  
- Micro e Minigeração (GD)  
- Compensação de Energia
